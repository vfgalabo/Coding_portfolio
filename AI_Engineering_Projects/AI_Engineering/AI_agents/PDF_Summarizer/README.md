# PDF Summarizer and Reader

This project is a Streamlit application that provides a powerful way to interact with PDF documents. It uses a Retrieval-Augmented Generation (RAG) system to allow users to upload a PDF and ask questions about its content.

## üìÑ Features

* **PDF Upload and Processing**: Users can upload any PDF file, which the application then processes by splitting it into manageable chunks.
* **Semantic Search**: A **FAISS vector store** and **Google Gemini embeddings** are used to perform a semantic search, retrieving the most relevant sections of the document to answer a user's query.
* **Advanced Q&A**: The application uses a **"refine" chain** to provide coherent and context-aware answers, ensuring that the model understands the document's content as a whole, rather than just isolated fragments.
* **Streamlit UI**: A simple and intuitive user interface built with Streamlit for a seamless user experience.

    ```bash
    pip install -r requirements.txt
    ```

3.  **Set Up API Key**:
    You will need a Google Gemini API key. It's recommended to store this in your environment variables.
    * Create a `.env` file or use your system's environment variables.
    * Alternatively, you can manually enter the key when prompted by the application.

4.  **Run the Application**:
    ```bash
    streamlit run pdf_app.py
    ```
